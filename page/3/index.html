DuckDuckGo for Safari

Apr 11th, 2012

DuckDuckGo is a popular alternative to Google Search. Unfortunately it’s not one of the search engines supported in Safari by default. Their instructions detail how to manually edit the /etc/hosts file to trick Safari into using DuckDuckGo instead of Yahoo.

The problem is that ceratin 3rd party applications, most notably the Cisco VPN Client, modify /etc/hosts in order to accomplish their purpose. In the case of the Cisco VPN Client, the custom DuckDuckGo/Yahoo entry is lost whenever this happens.

This Launch Daemon solves this problem by monitoring /etc/hosts, and adding the entry back if it’s removed. If you’ve had this problem, check out the project over on github and give it a try.
