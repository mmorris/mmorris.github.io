Vicious for Xcode

Jan 7th, 2013

I’d like to recommend a tool that I’ve been using over the past few weeks. Vicious is a plugin for Xcode that provides Vi keybindings in the text editor. When I first heard about it, I was skeptical that it could actually deliver on this promise, but I’ve been pleasantly surprised. I’ve used Vi for years, so I have a great deal of Vi muscle memory stored up in my fingers. Give it a try if you know Vi and have to work with Xcode.
