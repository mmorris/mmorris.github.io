Amazon Static Hosting Cost

Feb 4th, 2013

A little over a month ago, I moved this site to Amazon S3 Static Hosting. I’ve just received the bill for the first full month using the service. Here are the details.

This is for two websites. The “Reduced Redundancy” portion of the bill has nothing to do with the websites, but is for a bucket where I backup certain files.

So, the real price for hosting two websites on root domains was $1.1. Since I have very low traffic, most of the cost was for Amazon’s DNS service. You have to use their DNS service if you want to use a root domain, but if you keep the www subdomain, you could use your own DNS provider. In my case, if I weren’t using the root domain, my bill would only be about ten cents.

This seems inexpensive, but if you have more than a handful of websites with any meaningful traffic it could start to add up to more than a $9 per month shared hosting plan.
