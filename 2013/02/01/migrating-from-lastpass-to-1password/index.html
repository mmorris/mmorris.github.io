Migrating from LastPass to 1Password

Feb 1st, 2013

Today I decided to give 1Password a try. I’ve been a LastPass user for some time now, but I’ve always felt that their iPhone app left much to be desired. The 1Password apps are currently on sale for Macworld 2013, so I figured it’s as good a time as any to give it a shot. Here’s how I migrated my data from LastPass.

Export CSV from LastPass

The first step was to get the data out of LastPass. Click the LastPass button in Safari or Chrome, then click “Advanced”. You should see the following interface. Select “Export”.

This will give you a giant plaintext dump of your logins and passwords in CSV format. Save it to disk.

Importing the CSV Data to 1Password

Open 1Password on the Mac. From the File menu, choose “Import…”. You will see the following interface that begins the import process. Start by selecting “CSV or Delimited Text” from the dropdown.

On the next screen, choose the CSV file that you exported from LastPass, and click “Start Import”. Then, on the next screen, you just tell 1Password how to interpret the CSV data.

There are a few things you have to do on this screen to make things work right. The first line of the CSV file is header information for each column. So, select “Start import at line:1”. Also, there is a dropdown above each column that needs to be set so that 1Password knows how to interpret the colum. In the screenshot, I’ve set URL/Location, Username, Password, and Title.

In the next screen, you can review the entries. When you’re satisfied, just click “Import” and it will start populating the 1Password keychain with the lastpass CSV data.

Securely Remove the CSV File

This is important. You don’t want a plain-text copy of all your credentials sitting on disk.

On OS X, the rm command has an option to securely overwrite files before deleting them. The option you want is -P.

Overwrite regular files before deleting them. Files are overwritten three times, first with the byte pattern 0xff, then 0x00, and then 0xff again, before they are deleted.

This is an easy way to make reasonably sure that the data is no longer sitting unencrypted on your hard drive.

Finally

LastPass has served me well for a long stretch, but 1Password’s iOS and Mac apps are too compelling to ignore. I hope this walkthrough has been informative. Feel free to contact me if you find any errors.
